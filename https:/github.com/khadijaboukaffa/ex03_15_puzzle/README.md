# 15 Puzzle Solver with A*

## Description
This project solves a simple 15-puzzle instance using the A* search algorithm.

## Concepts used
- State space
- Neighbors
- Goal state
- A* search
- Manhattan distance heuristic

## What is measured
- Number of explored nodes before reaching the goal
- Number of moves in the solution

## Run
```bash
python3 puzzle15_astar.py
